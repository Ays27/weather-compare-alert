Weather Insight & Alert System
A web-based weather application that allows users to compare real-time weather conditions across different locations and receive alerts for significant weather changes such as extreme heat or rain.

Features
* Real-time weather data retrieval using an external API
* Weather comparison between multiple locations
* Displays temperature, wind, and environmental conditions
* Alert system for extreme weather changes
* Responsive and user-friendly interface

Tech Stack
* Nuxt.js
* TypeScript
* Weather API integration
* Vercel (Deployment)

 Live Demo
https://weather-compare-alert.vercel.app/

 Project Structure
* /pages – Application pages
* /components – Reusable UI components
* /composables – Logic for API calls and state handling
* /assets – Styles and static resources

* Working with external APIs and async data fetching
* Structuring a scalable frontend project
